Oraculo Stack

Habla natural, autentica una vez y deja que el agente decida.

Esta web crea usuarios y autentica contra oraculo_api, conserva la sesion y reenvia el token a oraculo_agente_ia. Si escribes una solicitud de prediccion con campos del Adult Income, el agente llama a tu API del modelo. Si haces una pregunta documental, responde desde su RAG con citas.

Como decide el agente

  • prediction Detecta campos del dataset y consulta tu modelo.
  • rag Busca informacion puntual y devuelve citas.
  • hybrid Combina prediccion y explicacion documental.
  • clarification Te pide los datos que faltan.
